# 06_data_structures — 03 Sets

---
# 🇬🇧 ENGLISH VERSION

## 📌 Description

This step introduces **sets**, a Python data structure that stores **unique** elements and does not keep a fixed order.

You will learn how to:

- Remove duplicates automatically
- Check membership quickly with `in`
- Use classic set operations:
  - Union (`|`)
  - Intersection (`&`)
  - Difference (`-`)
  - Symmetric difference (`^`)

The script also includes a small interactive CLI: **Unique Words Analyzer**.

---

## 🎯 Learning Objectives

- What sets are and how they differ from lists/tuples
- `add()`, `remove()`, `discard()`
- Membership test with `in`
- Set operations (union, intersection, difference)
- Practical use: unique words and common words between two sentences

---

## ▶️ How to Run

From inside the `06_data_structures/` folder:

```bash
python 03_sets.py
```
